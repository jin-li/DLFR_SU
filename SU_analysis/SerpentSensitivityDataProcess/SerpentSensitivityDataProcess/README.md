# Description
- Two C++ source files and one head file include the main code for this project.
- Directory "index" contains some index files used by MATLAB or C++.
- File "makefile" is the make file for nmake. nmake is a command tool provided by 
Microsoft in Visual Studio, which is similar to make in Linux. Please refer to NMAKE Reference 
(https://msdn.microsoft.com/en-us/library/dd9y37ha.aspx) for more details.
